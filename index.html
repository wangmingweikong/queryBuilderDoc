概述

QueryBuilder 是一个创建查询和过滤器的基于 jQuery 的 Bootstrap UI 组件。

  • 它可以用在搜索引擎页面、后台管理、数据查询模板等
  • 高度可定制和可组合其他第三方jQuery组件，如Bootstrap-typeahead，Bootstrap-select2等等
  • 允许用户选择输出结构的类型（SQL或者仅仅是JSON格式的结构），可以很容易地解析成SQL查询语句
  • 允许用户进行诸如多个模板嵌套查询模式
  • 一套伟大的插件，有一个完整的事件系统，甚至预留了支持更多的功能的接口
  • 提供复杂模板查询整合示例
